## Overview
This project analyzes how defensive contests affect basketball shooting outcomes.
The dataset was manually collected by watching five basketball games and recording
shot attempts, contest characteristics, and results.

## Repository Contents
- `data/`: Manually collected shot dataset
- `code/`: R scripts and R Markdown used to generate exploratory figures
- `report/`: Final project paper (PDF)
- `presentation/`: Project presentation slides
